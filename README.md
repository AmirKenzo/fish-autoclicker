# Fish Autoclicker

یک اسکریپت خودکار برای ماهیگیری در سانست

## نصب و راه‌اندازی

### 1. دانلود پروژه

ابتدا پروژه را از گیتهاب دانلود کنید:

```bash
git clone https://github.com/YOUR_USERNAME/fish-autoclicker.git
cd fish-autoclicker
```

یا می‌توانید فایل زیپ پروژه را دانلود و استخراج کنید.

### 2. نصب uv

برای مدیریت وابستگی‌ها و اجرای پروژه، نیاز به نصب `uv` دارید. 

لطفاً به [مستندات رسمی uv](https://docs.astral.sh/uv/getting-started/installation/) مراجعه کنید و `uv` را نصب کنید.

**برای Windows (PowerShell):**
```powershell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```


### 3. نصب وابستگی‌ها

پس از نصب `uv`، وابستگی‌های پروژه را نصب کنید:

```bash
uv sync
```

### 4. اجرای پروژه

برای اجرای پروژه از دستور زیر استفاده کنید:

```bash
uv run main.py
```

## استفاده

### شروع و توقف اسکریپت

- **F6**: برای شروع یا توقف اسکریپت استفاده می‌شود
  - با فشردن F6 اسکریپت شروع می‌شود
  - با فشردن مجدد F6 اسکریپت متوقف می‌شود

## پیکربندی

فایل `config.toml` را برای تنظیمات پروژه ویرایش کنید.

## هشدار و سلب مسئولیت

⚠️ **توجه**: هرگونه استفاده غیرمجاز از این پروژه، فروش آن و یا بن شدن در سرور به نویسنده مربوط نیست. استفاده از این اسکریپت به عهده خود کاربر است و نویسنده هیچ مسئولیتی در قبال عواقب استفاده از آن ندارد.
